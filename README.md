Search Engine

Описание проекта

Search Engine - это SpringBoot приложение, позволяющее осуществлять быстрый поиск страниц сайтов по заданным запросам.

Возможности

Индексация одного или нескольких сайтов  
Индексация отдельных страниц сайта  
Лемматизация текстов страниц  
Построение поискового индекса  
Поисковый запрос по строке текста  
Ранжирование результатов по релевантности  
Статистика по индексированным сайтам

Особенности

Использование ForkJoinPool при многопоточной индексации сайтов  
Лемматизация страниц с помощью Lucene Morphology   
Динамические userAgent и referrer  
Взаимодействие с веб-сервисом через REST API  
Поддержка mysql  
Логирование состояния приложения

Технологии

Java 21   
Spring Boot  
Spring Data JPA  
Hibernate  
MySQL   
JSoup   
Apache Lucene   
slf4j  
Lombok  
Maven

Основные команды API

GET /api/statistics - Возвращает общую статистику системы  
GET /api/startIndexing - Запускает процесс индексации всех сайтов  
GET /api/stopIndexing - Останавливает текущий процесс индексации  
POST /api/indexPage - Добавляет или обновляет отдельную страницу

Настройка

В application.yml указываем необходимые данные для подключение к MySQL и список сайтов для индексации

Запуск 

Приложение доступно по адресу:http://localhost:8080
